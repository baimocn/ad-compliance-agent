## 一、比赛核心定位

这是一场**「质量赛」**——不看你做了多少 Demo，只看你是否精准狙击了一个行业痛点。

> **核心目标：** 交付一个「能跑通商业闭环的行业 Agent」

### 评委在找什么

| 评分维度 | 具体要求 |
|---------|---------|
| **场景落地性** | 精准打中一个行业真痛点 |
| **技术创新性** | 架构精巧、有技术亮点 |
| **可复用性** | 能作为行业标杆推广复用 |
| **量化价值** | 说得清降本提效多少 |

### 评委反感什么

- 套壳聊天机器人
- 比谁 API 调得快、Demo 更花哨
- 跑不通真实业务、无法复用
- 说不清解决了谁的什么问题

## 二、Agent 还是玩具？

### 大模型 ≠ Agent

| 大模型（玩具态） | Agent（生产力） |
|:---|:---|
| 只会「思考」和生成文本 | **感知：** 看懂图表 / 听懂语音 |
| 没有记忆，容易「健忘」 | **规划：** 拆解复杂意图、定步骤 |
| 看不懂图表、调不了工具 | **记忆：** 记住行业上下文 |
| 答错了不会自我纠正 | **行动：** 调 API / 查库 / 跑代码 |

## 三、Agent 四要素评分标准

以 FinRobot（开源金融 Agent）为样本，逐项拆解：

### 要素①：领域知识接入 — ⚠️ 弱（你的机会）

**现状：** 只接入了实时数据（新闻、股价），没有接入行业私域知识库（法规、研报规范、会计准则）。

> **关键认知：** 调数据接口 ≠ 接入知识。这是最容易看错的地方。

**加分方向：** 接入行业 RAG 知识库

### 要素②：工具与系统打通 — ✅ 强（直接抄）

**优秀做法：**
- 只挂 4 个工具，职责单一，互不重叠
- 工具描述清晰（相当于写给 Agent 看的文档）
- 内置护栏：「只准用给你的函数」
- 任务完成后 TERMINATE，流程干净收尾

**加分方向：** 动作拆成精工具，少而准是第一原则

### 要素③：可靠性工程 — ⚠️ 偏弱（待补）

**缺失项：**

| 防线 | 状态 | 说明 |
|------|------|------|
| 数据溯源 | ✗ 缺 | 结论不标明出处，数字没挂「来自哪条新闻」 |
| 结构化输出 | ✗ 缺 | 输出是自然语言，无 Schema 约束 |
| 护栏 | △ 弱 | 仅「只用给定函数」，无拒答/越权拦截 |
| 人在回路 | ✗ 缺 | 一键全自动，无人确认环节 |

**加分方向：** 补上三道防线（溯源 + 结构化 + 人在回路）

### 要素④：评测与迭代 — ❌ 缺（送分题）

**缺失项：**
- 没有评测集
- 说不清准确率、错误率
- 说不清比人工快多少、省多少成本
- 没有量化指标 = 没有价值说明书

**加分方向：** 搭一个 30 题的小评测集，量化准确率/提速/成本

## 四、FinRobot 记分牌

| 要素 | 得分 | 给你的策略 |
|------|------|-----------|
| ② 工具打通 | ✅ 强 | 直接抄：动作拆成精工具 |
| ① 知识接入 | ⚠️ 弱 | 加分：接入行业 RAG |
| ③ 可靠性 | ⚠️ 偏弱 | 加分：补三道防线 |
| ④ 评测迭代 | ❌ 缺 | 送分：搭一个小评测集 |

**一句话策略：** 工具学它，知识/可靠性/评测超过它，作品就更扎实。

---

## 五、三个能直接拿走的招

1. **工具先做精** — 动作拆成职责单一、描述清楚的工具，加一句「只用我给的工具」
2. **把短板变亮点** — 溯源和评测工作量不大，却正压中「可靠性」和「量化价值」两项
3. **会读开源就是能力** — 拿四要素这把尺子去拆 GitHub 项目，哪些抄、哪些避、哪些补

## 六、评审规则（总分 110 分）

大赛评审采用 **线下现场评分（100分）+ 线上大众投票（10分）**

| 一级评分项 | 二级评分项 | 分值 |
|-----------|-----------|------|
| **🛠 技术创新性**（25分） | 技术架构与前沿性（10）<br>实现复杂度与规范性（10）<br>创新能力体现（5） | **25** |
| **📁 场景落地性**（30分） | 业务痛点匹配精准度（10）<br>业务闭环逻辑与落地可行性（10）<br>系统可扩展性与跨场景复用价值（10） | **30** |
| **🏅 作品完整性**（25分） | 功能模块完整性（10）<br>用户交互体验 UI/UX（10）<br>现场演示效果与系统稳定性（5） | **25** |
| **🗣 答辩表现**（20分） | 现场答辩逻辑与表达能力（10）<br>平台集成度与文档规范度（10） | **20** |
| **现场投票** | 现场用户投票 + 在线投票 | **10** |
