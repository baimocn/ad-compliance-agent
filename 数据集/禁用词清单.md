# 广告法禁用词清单

> 来源：基于《中华人民共和国广告法》(2015修订/2021修正) 第九条、第四条、第十七条、第十八条、第二十四条、第二十五条等条文整理
> 日期：2026.06.18
> 用途：广告法合规审查 Agent — 禁用词规则引擎数据源

---

## 一、极限用语 / 绝对化用语（第九条第(三)项）

### 1.1 "最"字系列
```
最、最佳、最好、最优、最大、最高、最低、最小、最强、最新、最先进、最流行、最受欢迎、
最畅销、最便宜、最低价、最前沿、最顶尖、最专业、最权威、最安全、最舒适、最时尚、
最经典、最完美、最先进、最全面、最先进、最出色、最优秀、最顶级、最火爆、最热销
```

### 1.2 "第一/首/唯一"系列
```
第一、唯一、首个、首选、首家、首屈一指、排名第一、销量第一、行业第一、全国第一、
全球第一、全网第一、品类第一、市场占有率第一、销量冠军、行业领先、遥遥领先、
独一无二、绝无仅有、硕果仅存、无与伦比、无可比拟、史无前例、前无古人
```

### 1.3 "顶级/极致"系列
```
顶级、极致、极品、绝品、巅峰、终极、至尊、至臻、王者、王牌、殿堂级、教科书级、
史诗级、航母级、重量级、现象级、标杆、天花板、封神、超神、登峰造极、出神入化
```

### 1.4 "国家级/世界级"系列
```
国家级、世界级、全球级、国际级、世界级品质、全球领先、世界一流、国际一流、
国家免检、国家认证（未标注具体认证号时）、中国名牌（无授权时）、驰名商标（用于广告宣传时）
```

### 1.5 "100%/绝对"系列
```
100%、百分百、完全、绝对、万能、通用、包治、包过、包赚、零风险、零缺陷、
零投诉、零差评、零失误、万无一失、稳赚不赔、有赚无赔
```

### 1.6 其他极限用语
```
全网最低、史上最低、击穿底价、亏本甩卖、跳楼价、吐血价、赔本价、秒杀价、
底价、成本价（非真实时）、买一送一（实际有附加条件时）、仅此一天、错过不再、
限时秒杀（无明确时限时）、最后XX件（虚假时）
```

## 二、虚假宣传 / 功效夸大（第四条、第二十八条）

### 2.1 医疗/健康功效类
```
根治、治愈、治疗、疗效、药到病除、包治百病、特效、神效、奇效、立竿见影、
一次见效、三天见效、七天见效、无效退款、无副作用、纯天然无害、安全无毒、
无任何风险、绝对不会过敏、替代药物、替代治疗、替代手术
```

### 2.2 美容/减肥功效类
```
永久美白、永久脱毛、永不反弹、一次瘦XX斤、快速瘦身、闪电瘦、秒瘦、
根治痘痘、根治色斑、根治皱纹、返老还童、逆龄、冻龄、不老、永葆青春、
丰胸、增大、增高（无批文时）、溶脂、排油、排毒（无科学依据时）
```

### 2.3 金融/投资收益类
```
保本保息、保本保收益、稳赚不赔、零风险高收益、躺着赚钱、年化收益XX%（承诺时）、
保证收益、固定收益、无风险套利、一本万利、一夜暴富、财务自由（承诺时）
```

### 2.4 教育/培训效果类
```
保过、包过、必过、包录取、包就业、100%通过率、签约保过、不过退款（无实际退款时）、
保证提分XX分、一个月提高XX分、速成、天才培养、学霸速成
```

### 2.5 房产增值承诺类
```
稳涨、必涨、只涨不跌、买到就是赚到（投资暗示时）、XX年回本、绝版、稀缺、
最后XX套、错过再无、升值潜力无限、投资回报率XX%
```

## 三、对比贬低 / 不正当竞争（第十一条、第十三条）

```
秒杀同行、吊打XX品牌、碾压竞品、完胜、全面超越、遥遥领先于XX、
比XX好10倍、XX根本不行、淘汰XX、替代XX（贬低竞品时）、
同行都不行（暗示竞品不好时）
```

## 四、资质/荣誉虚构（第十一条）

```
国家免检产品、质量免检、中国名牌、中国驰名商标（用于广告宣传）、
央视上榜品牌、央视推荐、XX部门推荐（无授权时）、XX专家推荐（无授权时）、
获XX奖（虚构或无法证实时）、XX认证（无具体编号时）
```

## 五、诱导/焦虑制造类

### 5.1 紧迫感虚假
```
仅限今天、最后一天、即将恢复原价、售完即止、限量XX件（虚假时）、
马上涨价、错过再等一年、最后机会、仅剩XX件（虚假库存时）
```

### 5.2 焦虑制造（教培/医疗/保险常用）
```
不学就落后、不买就后悔、别让孩子输在起跑线、再不XX就晚了、
别人家孩子都在学、你还在等什么、现在不XX以后XX
```

## 六、特殊行业禁用词

### 6.1 食品/保健食品
```
治疗、预防疾病、抗癌、降血压、降血糖、降血脂、减肥（普通食品时）、
增强免疫力（普通食品时）、延年益寿、防癌抗癌、清除自由基、排毒养颜、
替代药物、药食同源（夸大时）、祖传秘方、宫廷秘方
```

### 6.2 化妆品
```
医疗作用、治疗皮肤病、根治色斑、永久美白、科学验证（无验证时）、
临床验证（无临床数据时）、FDA认证（非真实时）、医生推荐（无授权时）、
无添加（未标注具体成分时）、纯天然（非纯天然时）、0刺激（未验证时）
```

### 6.3 酒类
```
消除紧张和焦虑、增强体力、有益健康、保健功能、药用价值、
治疗XX病、活血化瘀（非药品时）
```

### 6.4 烟草
```
任何暗示健康、低害、减害的用语均禁止
```
