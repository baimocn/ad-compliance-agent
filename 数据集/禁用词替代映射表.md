## 一、极限用语替代

| 禁用词 | 合规替代词（选一个用） | 说明 |
|--------|----------------------|------|
| 最好/最佳/最优 | 优质、优良、上乘、出色、出众 | 避免任何"最"字 |
| 最便宜/最低价 | 优惠价、特惠价、高性价比、超值 | 不承诺最低 |
| 最受欢迎 | 广受好评、深受欢迎、口碑良好 | 用"广受"替代"最" |
| 最畅销 | 热销、畅销、销量领先 | 去掉"最" |
| 最先进 | 先进、领先、前沿 | 去掉"最" |
| 最专业 | 专业、专注、深耕 | 去掉"最" |
| 最安全 | 安全可靠、安全放心 | 去掉"最" |
| 最新 | 全新、新款、新品 | 去掉"最" |
| 第一/排名第一 | 行业知名、广受认可、口碑良好 | 不可宣称排名除非有权威数据 |
| 销量冠军 | 热销产品、畅销款 | 不可宣称冠军 |
| 行业领先 | 专注XX领域、深耕XX行业 | 去掉排名暗示 |
| 唯一/独一无二 | 独特、特色鲜明、别具一格 | 不可宣称唯一 |
| 首选 | 推荐之选、值得选择 | 去掉"首" |
| 首家/首个 | 行业先行者、早期参与者 | 去掉"首" |
| 首屈一指 | 出类拔萃、品质出众 | 换用中性词 |
| 顶级/极致 | 高端、精品、精选、优选 | 不可宣称极致 |
| 极品/绝品 | 上品、佳品、优品 | 换用中性词 |
| 至尊/至臻 | 尊享、臻选、优选 | 换用中性词 |
| 巅峰 | 出色表现、优秀品质 | 不可宣称巅峰 |
| 天花板 | 高品质、高标准 | 不可宣称天花板 |
| 王者/王牌 | 旗舰、优选 | 不可宣称王 |
| 国家级 | 专业级、高品质 | 不可宣称国家级 |

| 100%/百分百 | 高品质保证、品质可靠 | 不可用百分比承诺 |
| 绝对 | 非常、十分、极为 | 换用程度副词 |
| 零风险 | 安全可靠、风险可控 | 不可宣称零风险 |
| 零差评 | 好评如潮、用户好评 | 不可宣称零差评 |
| 史无前例 | 重大优惠、大力度优惠 | 换用中性描述 |

| 爆款 | 热销款、人气款 | 去掉夸张表述 |
