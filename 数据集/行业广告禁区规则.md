# 行业广告禁区规则

> 来源：基于《广告法》第十七条、第十八条、第二十四条、第二十五条及行业专项规定整理
> 日期：2026.06.18
> 用途：广告法合规审查 Agent — 行业规则库数据源（子库 C）

---

## 一、食品行业

### 法律依据
- 《广告法》第十七条、第十八条
- 《食品安全法》第七十一条、第七十三条
- 《保健食品标识管理规定》

### 禁区规则

| 规则 | 说明 | 严重程度 |
|------|------|---------|
| 不得宣称疾病预防/治疗功能 | 食品广告不得涉及疾病预防、治疗功能 | 🔴 高 |
| 不得使用医疗术语 | 不得使用"处方""药用""治疗"等与药品混淆的用语 | 🔴 高 |
| 保健食品必须标注声明 | 必须标明"本品不能代替药物" | 🔴 高 |
| 不得暗示保健功能 | 普通食品不得暗示具有保健作用 | 🟡 中 |
| 不得利用广告代言人推荐保健食品 | 禁止利用个人推荐保健食品 | 🔴 高 |
| 不得声称全部营养 | 不得声称可以替代正常饮食 | 🟡 中 |
| 特殊医学用途食品须标注适用人群 | 必须在医生或临床营养师指导下使用 | 🔴 高 |

### 高频违规场景
```
❌ "这款酵素能排毒养颜、治疗便秘" → 食品宣称治疗功能
❌ "本品具有增强免疫力的功效" → 普通食品暗示保健功能
❌ "纯天然无任何副作用" → 食品暗示药用价值
✅ "本品选用优质原料，口感醇厚" → 合规描述
```

## 二、化妆品行业

### 法律依据
- 《广告法》第四条、第二十八条
- 《化妆品监督管理条例》第四十三条
- 《化妆品命名规定》

### 禁区规则

| 规则 | 说明 | 严重程度 |
|------|------|---------|
| 不得明示或暗示医疗作用 | 化妆品不是药品，不得宣称治疗效果 | 🔴 高 |
| 不得使用绝对化用语 | 不得使用"最""第一""国家级"等 | 🔴 高 |
| 不得使用他人名义保证效果 | 不得用医生/专家/患者名义推荐 | 🔴 高 |
| 普通化妆品不得宣称特殊功效 | 如美白、防晒、祛斑等需有特殊化妆品批文 | 🔴 高 |
| 涉及专利须标注专利号 | 宣称"专利技术"须标明具体专利号和种类 | 🟡 中 |
| 不得虚假标注成分 | 不得宣称含有实际不含有的成分 | 🔴 高 |
| 进口化妆品须标注原产国 | 必须标明原产国或地区 | 🟡 中 |

### 高频违规场景
```
❌ "这款面霜能根治痘痘、永久美白" → 化妆品宣称治疗+永久效果
❌ "经临床验证，100%有效" → 虚假临床验证+极限用语
❌ "医生推荐的祛痘神器" → 使用他人名义保证
✅ "含有烟酰胺成分，有助于提亮肤色" → 合规描述功效
```

## 三、教育培训行业

### 法律依据
- 《广告法》第二十四条
- 《关于进一步减轻义务教育阶段学生作业负担和校外培训负担的意见》(双减)
- 《民办教育促进法》

### 禁区规则

| 规则 | 说明 | 严重程度 |
|------|------|---------|
| 不得对升学/通过考试作保证性承诺 | 禁止"保过""包录取"等承诺 | 🔴 高 |
| 不得明示或暗示有考试机构参与 | 不得暗示与命题人员有关联 | 🔴 高 |
| 不得利用学术机构名义推荐 | 不得虚假使用高校/科研机构名义 | 🔴 高 |
| 学科类培训广告全面禁止 | 双减后义务教育阶段学科培训不得做广告 | 🔴 高 |
| 不得制造教育焦虑 | 不得使用"不学就落后""起跑线"等 | 🟡 中 |
| 不得虚假宣传师资力量 | 教师资质须真实可查 | 🔴 高 |
| 不得虚假宣传通过率 | 数据须真实有据 | 🔴 高 |

### 高频违规场景
```
❌ "签约保过，不过全额退款" → 对考试结果作保证性承诺
❌ "名师押题，命中率90%" → 虚假宣传+暗示与考试有关联
❌ "别让孩子输在起跑线" → 制造教育焦虑
✅ "系统化课程体系，助力学习提升" → 合规描述
```

## 四、金融理财行业

### 法律依据
- 《广告法》第二十五条
- 《关于进一步规范金融营销宣传行为的通知》
- 《商业银行理财业务监督管理办法》

### 禁区规则

| 规则 | 说明 | 严重程度 |
|------|------|---------|
| 不得对未来收益作保证性承诺 | 禁止"保本保息""稳赚不赔" | 🔴 高 |
| 须明确提示风险 | 投资有风险等风险提示必须显著 | 🔴 高 |
| 不得利用金融管理部门名义推荐 | 不得暗示银保监/央行背书 | 🔴 高 |
| 贷款广告须明示年利率 | 必须标注实际年化利率 | 🔴 高 |
| 保险广告须标明保险公司全称 | 必须标明承保公司名称 | 🟡 中 |
| 不得诱导过度负债 | 不得暗示"轻松借钱""零门槛" | 🟡 中 |
| 不得承诺固定收益（理财产品） | 收益表述须加"预期""历史"等限定词 | 🔴 高 |

### 高频违规场景
```
❌ "年化收益12%，保本保息" → 承诺固定收益+保本
❌ "零风险高收益，稳赚不赔" → 虚假承诺
❌ "央行推荐的理财产品" → 利用国家机关名义
✅ "历史年化收益率约5%-8%，投资有风险" → 合规描述
```

## 五、房地产行业

### 法律依据
- 《广告法》第二十六条
- 《房地产广告发布规定》

### 禁区规则

| 规则 | 说明 | 严重程度 |
|------|------|---------|
| 不得承诺升值或投资回报 | 禁止"买到就是赚到""稳涨" | 🔴 高 |
| 不得以项目到达某一参照物的所需时间表示距离 | 不得说"距天安门10分钟" | 🟡 中 |
| 不得含有风水等封建迷信内容 | 不得使用"风水宝地"等 | 🟡 中 |
| 面积须标明建筑面积或套内面积 | 不得模糊面积概念 | 🟡 中 |
| 价格须标明实际销售价格 | 不得使用虚假低价 | 🔴 高 |
| 预售须标明预售许可证号 | 必须有预售许可才能做广告 | 🔴 高 |

### 高频违规场景
```
❌ "买到就是赚到，每年稳涨20%" → 承诺升值
❌ "绝版地段，最后10套" → 极限用语+虚假紧迫感
❌ "风水宝地，旺财旺丁" → 封建迷信
✅ "交通便利，周边配套成熟" → 合规描述
```

## 六、医疗/药品行业

### 法律依据
- 《广告法》第十五条、第十六条
- 《医疗广告管理办法》
- 《药品广告审查办法》

### 禁区规则

| 规则 | 说明 | 严重程度 |
|------|------|---------|
| 处方药只能在专业期刊做广告 | 不得在大众媒体投放处方药广告 | 🔴 高 |
| 非处方药广告须标明OTC标识 | 必须标注"请按药品说明书或在药师指导下购买和使用" | 🔴 高 |
| 医疗广告须取得审查证明 | 须有《医疗广告审查证明》 | 🔴 高 |
| 不得保证治愈率或有效率 | 不得说"治愈率99%" | 🔴 高 |
| 不得利用患者形象做证明 | 不得使用患者前后对比照 | 🔴 高 |
| 不得与其他药品/医疗机构比较 | 不可贬低其他方案 | 🔴 高 |

## 七、酒类行业

### 法律依据
- 《广告法》第二十三条

### 禁区规则

| 规则 | 说明 | 严重程度 |
|------|------|---------|
| 不得暗示饮酒有消除紧张焦虑等功能 | 不得说"喝了就放松" | 🔴 高 |
| 不得出现驾驶车船飞机等画面 | 不得暗示饮酒后可以驾车 | 🔴 高 |
| 不得诱导未成年人饮酒 | 不得使用未成年人形象 | 🔴 高 |
| 须标注"过度饮酒有害健康" | 必须有健康提示 | 🟡 中 |
