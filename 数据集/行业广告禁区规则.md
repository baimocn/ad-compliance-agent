# 行业广告禁区规则

## 一、医疗行业

### 核心禁令
1. 禁止使用"治愈""根治""100%有效"等绝对化疗效用语
2. 禁止宣传治愈率、有效率等诊疗效果数据
3. 禁止利用患者名义或形象作证明（包括前后对比案例）
4. 禁止使用医学科研机构、学术机构、医疗机构的名义作推荐
5. 禁止以健康讲座、义诊咨询等形式变相发布医疗广告

### 常见违规点
- 以"免费体检""免费咨询"为名引流，实质为医疗广告
- 使用"三甲医院专家坐诊"等表述无法提供证明
- 美容机构发布"永久脱毛""一次见效"等功效承诺
- 口腔医疗机构宣传"种植牙终身质保"等无法兑现的承诺
- 减肥机构宣称"七天瘦十斤""不反弹"等虚假效果

### 抖音平台审核规则
抖音平台对医疗健康类广告实行严格的准入审核制度：
- 医疗机构需提供《医疗机构执业许可证》
- 医疗广告需取得《医疗广告审查证明》
- 禁止个人账号发布医疗诊断、治疗建议类内容
- 医美广告需特别标注"医疗美容有风险，选择需谨慎"
- 禁止在直播间进行疾病诊断、开具处方等医疗行为

### 小红书广告规范
小红书平台对医疗健康内容的审核要求：
- 禁止发布未经专业认证的医疗建议
- 医美分享笔记必须标注"医美风险提示"
- 禁止使用前后对比图暗示诊疗效果
- 医生认证账号需提供执业医师资格证
- 商业推广医疗服务需通过官方广告渠道

## 二、药品行业

### 核心禁令
1. 处方药不得在大众传播媒介发布广告
2. 禁止使用"无效退款""绝对安全""无毒副作用"等用语
3. 禁止与其他药品的功效和安全性进行比较
4. 禁止使用科研单位、学术机构、行业协会或专家、医生、患者的名义作证明
5. 禁止含有不科学的表示功效的断言或者保证

### 常见违规点
- 非处方药广告暗示可治疗多种疾病
- 药品广告擅自扩大适应症范围
- 利用明星代言药品广告
- 以"健康科普"名义变相推荐药品
- 中药宣称"祖传秘方""宫廷御药"等虚假宣传

## 三、保健食品行业

### 核心禁令
1. 禁止涉及疾病预防、治疗功能
2. 禁止声称或暗示为保障健康所必需
3. 禁止与药品、其他保健食品进行比较
4. 禁止利用广告代言人作推荐、证明
5. 必须显著标明"本品不能代替药物"
6. 禁止使用"最有效""最佳""第一品牌"等绝对化用语

### 常见违规点
- 保健食品宣称"治疗糖尿病""降血压"等疾病治疗功能
- 用"专家推荐""医生建议"等方式变相使用专业人士名义
- 以消费者证言、用户反馈形式变相使用代言人
- 直播带货中主播试吃并声称效果显著
- "蓝帽子"产品故意模糊保健食品与药品的界限

### 直播间合规要求
直播带货保健食品的特别注意事项：
- 主播不得对产品功效作任何断言或保证
- 必须完整展示保健食品标签和批准文号
- 必须口播"本品不能代替药物"警示语
- 不得使用"神奇效果""立竿见影"等夸张表述
- 不得组织"专家会诊""健康讲座"等形式进行营销

## 四、化妆品行业

### 核心禁令
1. 禁止宣传医疗作用或使用医疗术语
2. 禁止使用"药妆""医学护肤品"等概念
3. 禁止对化妆品的功效、安全性作断言或保证
4. 禁止使用"纯天然""无添加"等绝对化用语（除非能提供证明）
5. 禁止以名义使用他人名义保证或暗示效果使人误解

### 常见违规点
- 宣称"美白祛斑"却未取得特殊化妆品注册
- 使用"医用级""医美级"等暗示医疗效果的用语
- "七天美白""28天焕肤"等量化效果承诺无法证明
- 暗示产品具有"消炎""杀菌""修复受损肌肤"等医疗功效
- 化妆品广告使用"安全无毒副作用""孕妇可用"等安全性保证

### 化妆品广告中使用前后对比效果图的规范
化妆品广告中使用使用前后对比图需遵守以下规定：
- 对比图必须真实，不得经过过度修图或合成
- 必须标注"效果因人而异"等提示语
- 不得通过对比图暗示医疗效果
- 对比图需配合使用方法和周期的真实说明
- 特殊化妆品的功效宣称需有注册证支持

### 小红书种草笔记广告规范
小红书平台种草笔记的广告合规要求：
- 商业合作笔记必须标注"合作""广告"或"赞助"标识
- 禁止使用"神药""绝绝子""yyds"等夸张宣传用语
- 护肤产品分享需标注"个人肤质不同，效果因人而异"
- 禁止通过对比图、测评视频等方式暗示医疗效果
- 品牌方寄送的免费试用产品需在笔记中说明

## 五、食品行业

### 核心禁令
1. 禁止宣传治疗功效或疾病预防功能
2. 禁止使用医疗用语或易与药品混淆的用语
3. 禁止明示或暗示具有保健功能（除非取得保健食品批准）
4. 禁止虚假标注生产日期、保质期、营养成分等信息
5. 禁止使用"最健康""纯天然""有机"等无法证明的用语

### 常见违规点
- 普通食品宣称"增强免疫力""改善睡眠"等保健功能
- 食品广告暗示可以"替代药物""调理身体"
- "无糖""0卡"等标注与实际营养成分不符
- 儿童食品宣称"益智""增高""开发智力"等效果
- 绿色食品、有机食品等认证标志使用不实

## 六、教育培训行业

### 核心禁令
1. 禁止对升学、通过考试、获得学位学历等作出保证性承诺
2. 禁止明示或暗示有相关考试机构或其工作人员参与培训
3. 禁止利用科研单位、学术机构、教育机构、行业协会、专业人士、受益者的名义作推荐证明
4. 禁止"包过""保过""不过全额退款"等承诺性宣传
5. 禁止虚构师资力量、学员成绩、升学率等数据

### 常见违规点
- "命题组老师授课""阅卷人亲授"等虚假师资宣传
- "签约保过""不过退费"设置高额违约金或重重限制
- 冒用名校名义招生，实际并无合作关系
- 职业培训宣称"毕业即就业""月薪过万"等就业承诺
- 在线教育夸大AI教学效果，"智能因材施教"等无法兑现

### 抖音教育类广告规范
抖音平台教育类广告审核要点：
- 教育机构需提供办学许可证或营业执照
- 不得使用"保过""包过"等绝对化承诺用语
- 名师宣传需提供教师资格证和相关证明
- 不得与公务员考试、司法考试等法定考试挂钩
- 不得诱导未成年人过度消费或借贷培训

## 七、金融理财行业

### 核心禁令
1. 禁止对未来收益作出保证性承诺
2. 禁止明示或暗示保本、无风险或保收益
3. 禁止利用学术机构、行业协会、专业人士、受益者的名义作推荐证明
4. 必须对可能存在的风险及风险责任承担有合理提示或警示
5. 禁止使用"稳赚不赔""保本保息""零风险"等用语

### 常见违规点
- 理财产品广告只提收益不提风险，或风险提示极小
- 基金产品宣传"历史年化收益20%+"暗示未来收益
- 保险产品宣称"高收益""稳赚不赔"混淆保险与理财
- 股票荐股服务承诺"跟着买必赚""月收益30%"
- 以"财富自由""财务自由"等概念诱导非理性投资

### 金融理财产品广告合规要求
金融理财产品广告的合规要点：
- 风险提示必须醒目清晰，不得小于正文字号
- 不得使用"预期收益率""预计收益"等模糊表述误导
- 历史业绩不代表未来表现的声明必须突出展示
- 私募产品不得向不特定对象宣传推介
- 贷款类产品必须明示年化利率及综合资金成本

### 互联网金融广告平台规则
主流互联网平台对金融广告的特殊要求：
- 需取得金融监管部门的业务许可
- 贷款产品必须清晰展示年化利率
- 禁止诱导过度借贷或超前消费
- 禁止"校园贷""裸贷"等违法金融产品广告
- 不得使用"秒批""无门槛"等暗示降低风控标准的用语

## 八、房地产行业

### 核心禁令
1. 禁止升值或投资回报的承诺
2. 禁止以项目到达某一具体参照物的所需时间表示项目位置
3. 禁止违反国家有关价格管理的规定
4. 禁止对规划或建设中的交通、商业、文化教育设施及其他市政条件作误导宣传
5. 房源信息必须真实，面积必须标明建筑面积或套内建筑面积

### 常见违规点
- "买了必涨""抄底时机""投资洼地"等升值承诺
- "10分钟到市中心""步行5分钟到地铁站"等时间距离表述
- "学区房""名校入驻"等教育配套承诺，实际并未落实
- 以"内部认购""员工价"等名义虚假优惠
- 样板间与实际交付标准严重不符，隐瞒不利因素

### 房地产广告中关于面积和距离的表述规范
房地产广告中面积和距离表述的规范性要求：
- 面积必须明确标注是建筑面积还是套内建筑面积
- 公摊面积比例应清晰告知，不得隐瞒
- 距离表述必须使用实际长度单位（米/公里），不得用时间表示
- 交通配套需注明规划中/在建/已建成状态
- 周边商业、教育、医疗等配套需标明直线距离或实际路程

### 房产中介广告规范
房产中介机构发布房源广告的合规要求：
- 房源必须真实存在，价格必须真实
- 不得发布虚假低价房源吸引客户再推荐其他房源
- 房源图片必须为实景拍摄，不得使用效果图或他人房源图片
- 房屋面积、户型、朝向等信息必须与产权登记一致
- 不得隐瞒房屋重大瑕疵或不利因素

## 九、烟草行业

### 核心禁令
1. 全面禁止在大众传播媒介或公共场所发布烟草广告
2. 禁止向未成年人发送任何形式的烟草广告
3. 禁止利用其他商品或服务的广告宣传烟草制品
4. 禁止以公益广告形式变相宣传烟草
5. 禁止烟草企业发布迁址、更名、招聘等含有烟草产品信息的启事

### 常见违规点
- 烟草品牌赞助体育赛事、音乐节等活动变相宣传
- "电子烟""雾化棒"等新型烟草产品广告
- 通过KOL、网红在社交媒体"种草"烟草产品
- 烟草品牌周边产品（服饰、配件）广告间接宣传
- 以"文化传播""品牌推广"名义进行烟草营销

## 十、酒类行业

### 核心禁令
1. 禁止诱导、怂恿饮酒或宣传无节制饮酒
2. 禁止出现饮酒的动作
3. 禁止表现驾驶车、船、飞机等活动
4. 禁止明示或暗示饮酒有消除紧张和焦虑、增加体力等功效
5. 禁止向未成年人宣传酒类产品

### 常见违规点
- 广告中演员举杯畅饮、豪饮等饮酒动作
- 暗示"酒壮英雄胆""喝酒更有魅力"等诱导性内容
- 酒后驾驶、操作机器等危险画面
- "解酒护肝""养生酒""保健酒"等功效宣称
- 以青春、成功、时尚等概念绑定饮酒行为

## 十一、母婴用品行业

### 核心禁令
1. 禁止发布声称全部或部分替代母乳的婴儿乳制品广告
2. 禁止使用不满十周岁的未成年人作为广告代言人
3. 母婴食品不得宣称疾病预防或治疗功能
4. 婴儿用品不得使用"最安全""100%无害"等绝对化安全性用语
5. 不得在针对未成年人的大众传播媒介上发布不利于未成年人身心健康的广告

### 常见违规点
- 婴儿配方奶粉暗示"更接近母乳""优于母乳"
- 儿童保健品宣称"增高""益智""增强记忆力"
- 母婴用品使用"医用级""医院同款"等暗示医疗背书
- 纸尿裤"零过敏""绝对安全"等无法证明的安全性宣称
- 孕妇产品宣称"预防妊娠纹""产后快速恢复"等功效承诺

## 十二、网络游戏行业

### 核心禁令
1. 禁止在针对未成年人的大众传播媒介上发布不利于未成年人身心健康的网络游戏广告
2. 禁止诱导未成年人沉迷游戏或过度消费
3. 禁止宣传暴力、色情、赌博等不良内容
4. 游戏道具、虚拟货币广告不得暗示具有真实货币价值
5. 必须在显著位置标明"适度游戏益脑，沉迷游戏伤身"

### 常见违规点
- "首充6元得VIP""无限元宝"等诱导性充值广告
- 游戏宣传画面与实际游戏内容严重不符
- 以"赚钱""提现"等名义诱导用户下载游戏
- 暴力、血腥、恐怖元素过度渲染
- 针对未成年人的游戏攀比、炫耀消费营销

## 十三、通用禁令

### 所有行业通用禁区
1. 禁止使用"国家级""最高级""最佳""第一"等绝对化用语
2. 禁止使用或者变相使用国家机关、国家机关工作人员的名义或形象
3. 禁止使用国旗、国歌、国徽等国家标志
4. 禁止含有淫秽、色情、赌博、迷信、恐怖、暴力的内容
5. 禁止含有民族、种族、宗教、性别歧视的内容
6. 禁止损害国家尊严或利益，泄露国家秘密
7. 禁止妨碍社会公共秩序或违背社会良好风尚
8. 不得贬低其他生产经营者的商品或服务

### 自媒体平台广告通用规则
各主流平台对商业推广内容的共同要求：
- 商业广告必须有明确标识，不得伪装成自然内容
- 种草笔记、测评视频等有偿推广必须标注"广告""合作"
- 直播带货需遵守平台规范和广告法要求
- 不得使用机器刷量、虚假评论等方式伪造口碑
- 数据、效果宣称必须有客观依据
